# ConsoleApp1 -  консольное приложение на C#

Простой проект для демонстрации работы с SQLite из C#.

## Что умеет приложение

- Создаёт базу данных SQLite (файл `demo.db`)
- Создаёт таблицу `users`
- Добавляет тестовую запись
- Выводит список всех пользователей в консоль

## Как запустить

1. Установите [.NET 6.0 или выше](https://dotnet.microsoft.com/download)
2. Склонируйте репозиторий: git clone https://github.com/DmitriyGusev1991/ConsoleApp1.git
3. Перейдите в папку проекта и запустите: dotnet run

## Используемые технологии

- C# / .NET [ваша версия, например 6.0]
- SQLite
- Microsoft.Data.Sqlite

## Цель проекта

Pet-проект для изучения работы с базами данных из C#.
